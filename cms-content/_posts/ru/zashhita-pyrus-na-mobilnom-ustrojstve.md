Мы добавили опцию установки пин-кода для входа в мобильное приложение Pyrus. Это дополнительная мера для обеспечения безопасности вашего аккаунта. Теперь, даже если кто-то получит доступ к вашему телефону, он не сможет открыть на нём приложение, не зная пароля.

Для установки пин-кода откройте вкладку «Ещё» и выберите пункт «Настройки входа». Включите опцию «Вход по пин-коду» и придумайте четырёхзначный пароль.

![](phone_iphone13midnight_portrait-548x1024.webp)

Кроме пин-кода можно настроить вход по биометрическим данным: по Face ID или Touch ID. Для этого после установки пин-кода включите опцию «Вход по биометрии». После этого телефон будет узнавать вас по лицу или отпечатку пальца, и вам не придётся каждый раз вводить пин-код. 

![](phone2_iphone13midnight_portrait-1024x792.webp)

Функция работает в приложениях для платформ: iOS и Android.
